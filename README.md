# Magento2 Quick Cart
Free magento 2 extension to add to cart using ajax | Nadeem Khan

<img src="https://i.ibb.co/nnVLZnV/Magento2-2.png" alt="Magento2-2" border="0">

## Features:

1. Easy to use
2. Looks like core functionality of Magento (at backend as well as frontend)
3. Unencrypted code for easy customization

# Installation Instruction

* Copy the content of the repo to the <b>app/code/Magelumen/DeleteOrders/</b> folder
* Run command:
<b>php bin/magento setup:upgrade</b>
* Run Command:
<b>php bin/magento setup:static-content:deploy</b>
* Now Flush Cache: <b>php bin/magento cache:flush</b>
